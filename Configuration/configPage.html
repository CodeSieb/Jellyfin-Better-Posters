URL Patterns Reference

The plugin builds a btttr.cc URL from your toggle combination. Each row below maps to a real btttr.cc endpoint — all seven were verified to return HTTP 200 on 2026‑06‑24 against tt0111161 (Shawshank Redemption). The base URL is https://btttr.cc/poster-X/{imdb | tmdb}/poster-default/{ID}.jpg where the X suffix encodes the overlay set and an optional query string handles trend suppression, language, and rating source.
Toggles URL pattern
Trend on (default), Genre off, Rating off, Quality off, Age off https://btttr.cc/poster-n/imdb/poster-default/{ID}.jpg
All overlays off (plain English) https://btttr.cc/poster-n/imdb/poster-default/{ID}.jpg?tag=none
Genre on, Rating off, Trend off https://btttr.cc/poster-g/imdb/poster-default/{ID}.jpg?tag=none
Rating on, Genre off, Trend off https://btttr.cc/poster-r/imdb/poster-default/{ID}.jpg?tag=none
Quality on, Genre off, Rating off, Trend on https://btttr.cc/poster-nq/imdb/poster-default/{ID}.jpg
Age on, Genre off, Rating off, Trend off https://btttr.cc/poster-na/imdb/poster-default/{ID}.jpg?tag=none
Genre on, Quality on, Age on, Trend on (Rating on or off — "all combined") https://btttr.cc/poster-gqa/imdb/poster-default/{ID}.jpg
Path letter rule: g — Genre is on (Rating renders inside the genre strip so we omit r). r — Rating is on but Genre is off (rating stands alone). n — neither Genre nor Rating on. Then +q if Quality tags on, +a if Age rating on. The path always carries at least one of those three letters so the URL never collapses to a bare poster/.
Query string: tag=none is added only when Trend Tags is off; lang=xx is added when Language is not English (e.g. lang=es, lang=fr); rs=xx is added when Rating Source is not “Average” (IM / TM / RT / MC / LB / RE).
